Broj generiranih ulaznica: 0

Kako se radi ulaznica:

Da biste generirali ulaznicu, slijedite ove korake:

  1. Dobijanje JWT Tokena: Prvo, pošaljite POST zahtjev na https://dev-0m6ezck5okgu8bj4.us.auth0.com/oauth/token kako biste dobili JSON Web Token (JWT).
  2. Generiranje ulaznice: Nakon što dobijete token, pošaljite POST zahtjev na http://localhost:3000/api/generateTicket. U zahtjevu dodajte token u **Authorization** zaglavlje koristeći format **Bearer Token**.
  3. Tijelo zahtjeva: U tijelu zahtjeva pošaljite JSON podatke koji sadrže:
    • OIB (VATIN): Jedinstveni identifikacijski broj za fizičke ili pravne osobe.
    • Ime: Ime osobe koja će koristiti ulaznicu.
    • Prezime: Prezime osobe koja će koristiti ulaznicu.
  4. Ograničenje ulaznica: Za jedan OIB, dozvoljeno je generirati do tri (3) ulaznice.
  5. Generiranje QR Koda: Nakon što je ulaznica spremljena, generirat će se QR kod koji sadrži URL stranice s informacijama o ulaznici.
Podatke o ulaznici korisnik može pogledati na http://localhost:3000//api/tickets/:ticketId.
